Estrutura Básica do HTML

( Voltar )

O nome index é o padrão, indicando que é o início do site, é uma convenção.

O html consiste em 3 partes (tags):

  • A tag html comporta todo o site html.
  • Já a tag head contém o cabeçalho do site,
    contendo tudo que vai ser pré carregado para o site,
    sem estar visível para o usuário.
  • A tag body, onde estão todos os componentes visíveis para o usuário no navegador.

Atributos de Tags

( Voltar )

Os atributos são propriedades que uma tag pode ter, alguns são genéricos, que toda tag tem, e tem alguns específicos para cada tag.

Genéricos:

  • id -> Serve para identificar um elemento específico para ser manipulado no Javascript.
  • style -> Normalmente usado em CSS. Implementação / Formatação do HTML através de CSS.
  • class -> Normalmente usado em CSS. Utilizado para padronizar tags.

Específico

( Voltar )

  • A tag input tem o atributo type, que pode ser text, number, color, entre outros. Sendo atributos específicos de uma tag.
  • A tag img tem o atributo src, para indicar o caminho da imagem que o navegador deve mostrar.

Textos

( Voltar )

São as tags específicas para as tipografias que o navegador vai interpretar e construir para o usuário.

Principais:

  • h1, h2, h3, h4, etc -> São as tags para títulos, subtítulos…

  • blockquote -> Tag utilizada para
    citações, observações, algo destacado em um bloco
    .

  • tag p -> Tag de parágrafo, onde cada quebra de linha, com .

    Irá iniciar um novo parágrafo.

  • tag u -> Sublinhado.
  • tag i -> Itálico
  • tag mark -> Marcado
  • sup -> adiciona o texto na parte superior da linha.

O navegador dá estilos padrões para as tags, margem, tamanho, fonte, etc. É possível alterar via CSS.

Em aulas de CSS, será visto responsividade, onde o navegador ajusta o tamanho do site de acordo com o tamanho da tela.

w3schools.com tem todas as tags html documentadas para consulta e entendimento de uso.

Links

( Voltar )

Links vão ancorar páginas, para que possamos ir de uma página à outra.

É utilizado a tag a que ancora um caminho utilizando o atributo href.

Para que, ao clicar em um link, seja aberto uma nova aba, é utilizado o atributo chamado target.

Ele tem como valores: _blank, _parent, _self e _top.

Hoje em dia não é utilizado mais _parent e _top, somente _blank e _self.

  • _self é o comportamento padrão, ao clicar no link, ele abre na aba atual.
  • _blank abre em uma nova aba.
  • Se for clicado segurando ctrl, abre sempre em nova aba.
  • Se for clicado segurando shift, abre sempre em nova janela.

Temos na tag a o atributo title, onde ao deixar o mouse em cima do link, retorna um tooltip com o texto definido no title.
